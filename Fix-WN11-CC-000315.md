# WN11-CC-000315 - Disable "Always Install with Elevated Privileges"

## Manual Remediation

### Step 1: Open Local Group Policy Editor

1. Open the Windows Search bar.
2. Type:

```text
gpedit.msc
```

3. Press **Enter**.

---

### Step 2: Configure the Computer Policy

Navigate to:

```text
Computer Configuration
└─ Administrative Templates
   └─ Windows Components
      └─ Windows Installer
         └─ Always install with elevated privileges
```

Set the policy to:

```text
Disabled
```

---

### Step 3: Configure the User Policy

Navigate to:

```text
User Configuration
└─ Administrative Templates
   └─ Windows Components
      └─ Windows Installer
         └─ Always install with elevated privileges
```

Set the policy to:

```text
Disabled
```

---

### Step 4: Apply the Changes

Open **Command Prompt** as Administrator and run:

```cmd
gpupdate /force
```

---

### Verification

Confirm that **Always install with elevated privileges** is set to **Disabled** in both:

* Computer Configuration
* User Configuration

Re-run the compliance scan to verify that **WN11-CC-000315** is no longer reported as a finding.
