Manual Remediation.

 └─ Administrative Templates
     └─ Windows Components
         └─ Windows Installer
             └─ Always install with elevated privileges

Set to : Disabled

User Configuration
 └─ Administrative Templates
     └─ Windows Components
         └─ Windows Installer
             └─ Always install with elevated privileges

Set to : Disabled
