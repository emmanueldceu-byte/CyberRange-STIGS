# WN11-CC-000110 - Printing Over HTTP Must Be Prevented

## Overview

This STIG requires Windows 11 systems to prevent printing over HTTP.

**Required Configuration:**

* Policy: Turn off printing over HTTP
* State: Enabled

---

# Manual Remediation

## Step 1: Open Local Group Policy Editor

Open the Windows Search bar and type:

```text
gpedit.msc
```

Press **Enter**.

---

## Step 2: Configure the Policy

Navigate to:

```text
Computer Configuration
└─ Administrative Templates
   └─ System
      └─ Internet Communication Management
         └─ Internet Communication settings
            └─ Turn off printing over HTTP
```

Configure the policy as:

```text
Enabled
```

Click:

```text
Apply
OK
```

---

## Step 3: Apply Group Policy

Open Command Prompt as Administrator and run:

```cmd
gpupdate /force
```
