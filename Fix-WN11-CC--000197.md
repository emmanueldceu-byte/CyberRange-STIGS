# WN11-CC-000197 - Microsoft Consumer Experiences Must Be Turned Off

## Overview

This STIG requires Microsoft Consumer Experiences to be disabled to prevent Windows from automatically installing consumer applications, suggested apps, and promotional content.

**Required Configuration:**

* Policy: Turn off Microsoft consumer experiences
* State: Enabled

---

# Manual Remediation

## Step 1: Open Local Group Policy Editor

Open the Windows Search bar and type:

```text
gpedit.msc
```

Press **Enter**.

---

## Step 2: Configure the Policy

Navigate to:

```text
Computer Configuration
└─ Administrative Templates
   └─ Windows Components
      └─ Cloud Content
         └─ Turn off Microsoft consumer experiences
```

Configure the policy as:

```text
Enabled
```

Click:

```text
Apply
OK
```

---

## Step 3: Apply Group Policy

Open Command Prompt as Administrator and run:

```cmd
gpupdate /force
```

---

which satisfies the STIG requirement.
